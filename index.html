How to add Audio?

Use audio Tag

Purpose:Embeds sound (audio) content in a webpage.

  1. controls: Display play/pause, volume controls.
  2. autoplay: Automatically starts playing the audio.
  3. loop: Loops the audio after finishing.
  4. preload: Specifies wheather to load the audio before playing

How to add video?

Use of video tag

attributes:

  1. controls: Shows playback controls(play/pause, fullscreen)
  2. autoplay: Video starts playing automatically.
  3. loop: Repeats the video after it ends.
  4. muted: Starts video muted
  5. poster: Displays an image before the video plays.
  6. preload: Defines how the video should load (auto, metadata, none).

How to add Youtube video to Webpage

Using ifreame Tag Purpose: Embeds external content (web pages, maps, other media) into webpage

Attributes:

  1. src: URL of the content to be embedded.
  2. width and height: size of the iframe
  3. frameborder: Controls wheather the iframe has a border
  4. allow: Defines permissions (fullscreen, autoplay)
